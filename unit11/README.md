## 🎮 **게임 방법**

1. **게임 시작**: "Game Start" 버튼 클릭
2. **단어 확인**: 화면 상단의 한글 단어와 품사 확인
3. **UFO 격추**: 날아오는 UFO 중 정답을 FIRE 버튼으로 격추
4. **점수 획득**: 정답 시 +10점, 오답 시 -5점
5. **보너스 게임**: 정답 후 타이핑 게임으로 +5점 추가 획득
6. **TTS 활용**: 스페이스바로 발음 듣기
