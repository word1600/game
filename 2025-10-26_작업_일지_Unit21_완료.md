**문제**: 틸드 문자(~)가 포함된 한글 표현 검증 실패
```
Line 11: "~을 상대로"

Line 31: "~에 근거하여"
```

**해결**: `enhanced_validator.py` 수정

```python
# 변경 전
r'^[가-힣a-zA-Z0-9\s,()!]+$'

# 변경 후
r'^[가-힣a-zA-Z0-9\s,()!~]+$'
```
